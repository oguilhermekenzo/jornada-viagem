# ✈️ Jornada Viagem

Um projeto de interface web focado em responsividade e design de layouts, desenvolvido durante o **Curso de HTML e CSS: responsividade em layouts**.

🔗 **[Acessar o projeto online](https://jornada-viagem-one.vercel.app)**

## 🎯 Sobre o Projeto

O **Jornada Viagens** é uma página voltada para o setor de turismo. O principal objetivo deste projeto foi aplicar conceitos modernos de CSS para construir uma interface fluida. Utilizando a abordagem **Mobile First**, o layout foi garantido para se adaptar perfeitamente a diferentes tamanhos de tela (dispositivos móveis, tablets e desktops), proporcionando uma ótima experiência de usuário e melhor performance.

## ✨ Destaques do Desenvolvimento

Durante a construção deste layout, diversas técnicas de estilização foram aplicadas para refinar o design:

- **Metodologia Mobile First:** O CSS foi estruturado inicialmente para telas menores e, através de Media Queries (ex: `min-width: 768px`), escalado progressivamente para resoluções maiores, garantindo um código mais limpo e otimizado.
- **Layouts Bidimensionais:** Uso avançado de CSS Grid para estruturar as seções, mantendo o alinhamento correto entre textos e imagens em qualquer dispositivo.
- **Tratamento de Imagens:** Aplicação de `object-fit` para garantir que as fotos preencham seus containers de forma harmônica, sem distorções, independentemente da altura do texto lateral.
- **Formulários Customizados e Limpos:** - Estilização minimalista de campos de entrada (inputs), utilizando apenas bordas inferiores e transições suaves com `:focus` e `::placeholder`.
  - Remoção de estilos indesejados padrão dos navegadores, como as setas de incremento em inputs do tipo numérico e o contorno (outline) de seleção.
  - Limite seguro de área de digitação em `textareas` utilizando `max-width` e `max-height` para evitar que o redimensionamento manual quebre o layout da página.
- **Variáveis CSS:** Utilização de propriedades customizadas (como `--secondary-color`) para manter a consistência visual e facilitar a manutenção do código ao longo do projeto.

## 🚀 Tecnologias Utilizadas

- **HTML5** (Estruturação semântica)
- **CSS3** (Grid Layout, Media Queries, Mobile First)
